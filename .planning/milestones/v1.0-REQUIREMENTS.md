# Requirements Archive: v1.0 Initial Release

**Archived:** 2026-04-02  
**Status:** SHIPPED  
**Result:** 14 / 14 v1 requirements completed

## Validated Requirements

### Lock Semantics

- [x] `LOCK-01` 支持在单箱、双箱、木桶、潜影盒等目标上创建主锁
- [x] `LOCK-02` 只有所有者或管理员绕过可以继续添加扩展授权牌
- [x] `LOCK-03` 双箱两半统一视为同一锁目标
- [x] `LOCK-04` 锁牌编辑保留系统结构行

### Protection

- [x] `PROT-01` 未授权玩家使用受保护目标时会被稳定拦截
- [x] `PROT-02` 未授权玩家打开受保护容器时会被稳定拦截
- [x] `PROT-03` 未授权玩家不能破坏受保护方块本体或他人锁牌
- [x] `PROT-04` 自动化、爆炸、活塞和流体不能绕过保护

### Operations

- [x] `OPS-01` 服主可以通过 `/signlock reload` 与 `/bl info` 管理和检查状态
- [x] `OPS-02` 锁主人可以通过 `/signlock add` / `/signlock remove` 管理授权
- [x] `OPS-03` 玩家身份缓存与改名解析保持一致

### Localization & Release

- [x] `REL-01` 默认 `config.yml`、`plugin.yml` 和提示文案可用于中文服务器
- [x] `REL-02` 仓库提供可执行的手测与发布检查文档
- [x] `REL-03` 项目在当前 Spigot / Paper / Folia 目标环境下可成功构建
